# Used Car Price Prediction

本專題使用了二手車網站的價格來為使用者預估特定車種的二手價格。使用者首先需 輸入品牌編號，再輸入系列編號，即開始查詢該系列的車輛資料 (價格、出產年份及里程數 )。 待 資料收集完畢後， 建立該系列的價格模型，並讓使用者輸入車輛的出產年份及里程數以獲得預估的價格，同時也輸出資料集的價格分布圖，並標示預估車輛在圖中相應的位置。建立完一組車系模型後，可重複預估此車系的價錢，但選擇結束該輪預估後，模型資料即被刪除，下次需要預估同車系時需從收集資料的重新進行。當輸入中
出現錯誤，如不存在的車輛編號、負里程數等，會出現錯誤提示告知使用者，並提示其重新輸入。

此專案主要使用了 selenium模組進行爬蟲。模型預測則選擇 sklearn 中的KNeighborsRegressor。品牌與車系使用 SQLite作為資料庫儲存。
使用tabulate模組輔助呈現；車輛結果分布圖則由 matplotlib 繪製。
