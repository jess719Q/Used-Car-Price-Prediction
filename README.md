# Used Car Price Prediction Through KNN

本專案使用二手車網站的資訊，透過 KNN 模型根據車輛的 購入年份及里程數預測二手車價格。資料收集透過 selenium 模組進行網頁爬蟲。

### 使用方法

1. 使用 load_data.py 下載或更新品牌及車系資料。
2. 由 predict.py 查詢車輛價格: 首先需輸入品牌編號，再輸入系列編號，即開始查詢該系列的車輛資料 (價格、出產年份及里程數 )。 待資料收集完畢後， 建立該系列的價格模型，並讓使用者輸入車輛的出產年份及里程數以獲得預估的價格，同時也輸出資料集的價格分布圖，並標示預估車輛在圖中相應的位置。建立完一組車系模型後，可重複預估此車系的價錢，但選擇結束該輪預估後，模型資料即被刪除，下次需要預估同車系時需從收集資料的重新進行。當輸入中出現錯誤，如不存在的車輛編號、負里程數等，會出現錯誤提示告知使用者，並提示其重新輸入。
